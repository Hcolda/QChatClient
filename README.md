If you want the server, see [QingLiaoChatServer](https://github.com/Hcolda/QingLiaoChatServer)

## Build
### Essential tools
1. [vcpkg](https://github.com/microsoft/vcpkg)

### Build with cmake
```cmd
cmake -S . -B build
cmake --build build --config Release
```
